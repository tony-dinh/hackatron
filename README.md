## Overview
Hackatron is a multiplayer deathmatch action game.

## Installation
Clone the project:
```
git clone https://github.com/tony-dinh/hackatron.git
```

Install dependencies:
```
cd hackatron
npm install
```

Spin up a local server:
```
node server.js
```

Open `localhost:8080` in your browser & enjoy! :)
